SafeTrack HSE Portal Site Plan

Site Name

SafeTrack HSE Portal

This name was selected because the website focuses on helping organizations track workplace safety, report incidents, and access useful health, safety, and environmental resources.

Optional Domain: safetrackhse.com

Site Purpose

The SafeTrack HSE Portal will provide a simple online platform for workplace safety information, incident reporting, and HSE resource sharing. The site will help users learn about safety practices, browse safety resources, and submit incident reports through an online form.

Scenarios

  • How can an employee quickly report a workplace incident or unsafe condition?
  • Where can a supervisor find simple safety resources for toolbox talks and staff training?
  • What safety topics should a company focus on to improve workplace compliance?

Color Scheme

Primary Color: #0f4c5c — Used for headings, header, footer, and buttons.
Secondary Color: #f4a261 — Used for accents, highlights, and call-to-action elements.
Background Color: #f7f9fb — Used for the page background.
Text Color: #222222 — Used for main body text.

Typography

Roboto will be used for headings, body text, buttons, labels, and navigation links. It was selected because it is clean, readable, and suitable for a professional HSE website.
